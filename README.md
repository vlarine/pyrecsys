Installation
------------

To install from [PyPI](https://pypi.python.org/pypi/recsys/), use pip:

    $ pip install recsys

Or clone the repo:

    $ git clone https://github.com/vlarine/pyrecsys.git
    $ cd pyrecsys
    $ pip install -r requirements.txt
    $ pip install .
